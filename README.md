# 医联体培训管理系统

基于 Vanilla JS 构建的医联体培训管理系统高保真原型，无需构建工具，纯静态部署。

## 在线预览

> 部署于 GitHub Pages，点击下方链接直接访问：  
> **https://YOUR_USERNAME.github.io/YOUR_REPO/**

## 角色体验

| 角色 | 说明 |
|---|---|
| 医联体超级管理员 | 查看全量数据，管理所有培训计划、录像、档案 |
| 医院/科室讲师（张志国·主任医师） | 管理自己的培训计划，主持培训会议室，查看自己的录像 |
| 普通医护人员 | 参与培训，查看案例，回看往期录像 |

## 功能模块

- 培训计划管理（角色权限隔离）
- 培训会议室（案例实时展示 / 当前讲解标注 / 侧边案例抽屉）
- 往期培训录像（自动归档 / 章节跳转 / 关联案例）
- 培训档案归档 / 评价管理 / 证书管理
- 签到管理 / 数据看板

## 本地运行

```bash
node server.js
# 访问 http://localhost:5500
```
